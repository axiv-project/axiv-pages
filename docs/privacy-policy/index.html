プライバシーポリシー・免責事項

本プライバシーポリシーは、スマートフォンアプリ「AXiV」（以下「本アプリ」）における、利用者情報の取り扱いについて定めるものです。

1. 取得する情報

本アプリは、利用者の氏名、メールアドレス、住所、電話番号等の個人情報を、開発者が特定の利用者と結び付けて取得・保存しません。

また、本アプリは、広告識別子（IDFA）等のトラッキング目的の識別子を利用しません。

2. 収集・利用（分析等）について

本アプリは、アプリ内の入力内容や利用履歴等を、開発者が特定の利用者と結び付けて収集・保存し、分析・マーケティングに利用することはありません。

3. 外部サービス（第三者提供）

本アプリは、原則として、利用者情報を外部の第三者へ提供しません。

ただし、OS（iOS）や端末が提供する標準機能（例：通信、通知、設定等）を利用する場合があり、当該機能の提供者（Apple等）による取り扱いについては、 各提供者のプライバシーポリシーをご確認ください。

また、インターネット通信の過程で、通信事業者・端末/OS・ホスティング事業者等により、IPアドレス等の情報が記録される場合があります。

4. お問い合わせ

本ポリシーに関するお問い合わせは、以下までご連絡ください。

連絡先: axiv.app.contact@gmail.com

なお、メールでお問い合わせいただいた場合、開発者は、お問い合わせへの回答のために、メールアドレスおよびお問い合わせ内容を取り扱う場合があります。

5. 免責事項

本アプリの利用により生じた損害（データの消失、端末の故障、その他一切の不利益）について、AXiV Projectは、理由の如何を問わず一切の責任を負いません。 本アプリは現状有姿で提供されるものであり、その正確性や安全性を保証するものではありません。

6. 改定

本アプリの機能追加や法令変更等に応じて、本ポリシーを改定する場合があります。
